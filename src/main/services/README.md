# Services

This directory contains the core business logic of the application, separated into modules.

## Modules

- `streaming/`: Logic for RTMP streaming.
- `recording/`: Logic for screen and audio recording.
- `scenes/`: Logic for managing scenes and sources.
- `audio/`: Logic for audio mixing and device management.
- `tokens/`: Logic for managing authentication tokens.
- `storage/`: Logic for file system persistence.
